# 기술 스카우팅 스킬

## 실행 순서

1. 탐색 대상 라이브러리/기술 확인
2. WebSearch로 조사:
   - GitHub 저장소 (별, 최근 커밋, 이슈)
   - npm 주간 다운로드 수
   - 번들 크기 (bundlephobia)
   - 알려진 이슈/breaking changes
3. GitHub README 분석 (주요 기능, 설치법)
4. 프로젝트 호환성 확인:
   - Next.js 15 App Router 호환?
   - React Server Component 호환?
   - TypeScript 타입 포함?
   - Tailwind CSS v4 호환?
5. 추천/비추천/대안 제시
